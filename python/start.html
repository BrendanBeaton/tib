4.3. Everyones first program in Python¶

print("Hello World!")

Hello World!

Now that we’ve got that out of the way, let’s first treat Python as just a calculator.

4.4. Basic arithmetic operations¶

4.4.1. Addition¶

1 + 9

10

4.4.2. Subtraction¶

1 - 9

-8

4.4.3. Multiplication¶

2 * 20

40

4.4.4. Division (including integer division)¶

Standard division uses a single /

20 / 3

6.666666666666667

Integer division uses //

20 // 3

6

4.4.5. Exponents / Powers¶

2 ** 4

16

4.4.6. Roots¶

4 ** (1 / 2)

2.0

4.4.7. The math module¶

More sophisticated mathematical routines are included in the math module. We will discuss modules later.

4.4.8. Exercises¶

1. Do the hello world example yourself.

2. Compute the square root of 16 (i.e. \(\sqrt 16\)). What type is 16? What type is the result?

3. Order of operations rules. Compute the following expressions

(10 + 2) * 2

and

10 + 2 * 2
